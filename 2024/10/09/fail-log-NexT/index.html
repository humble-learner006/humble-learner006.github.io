使用NexT主题时不报错但是显示错误（破案：不是最新的next官网）

follow the instructions from official website
the version in the instructions is not the latest, and instruction is also not latest

the only thing appear on the localhost is pure text as follow

it is some problem relate to swig
few steps to troubleshoot:

Check Data Availability: Ensure that page.posts contains the expected data. If page.posts is empty or not defined, the loop will not render any posts.

Verify Template Paths: Make sure the paths to _layout.swig, _macro/post.swig, and _macro/sidebar.swig are correct and the files exist.

Debugging Output: Add some debugging output to check if the blocks are being executed correctly. For example, you can add some static text inside the blocks to see if they are rendered.

Check Swig Configuration: Ensure that Swig is configured correctly in your project. Sometimes, issues can arise from incorrect Swig settings.
